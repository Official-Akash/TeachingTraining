CSS Color format

  • hex-hexadecimal-#rrggbb;-(0-f)-#ff0000;-(red)
    1. #000000-black
    2. #ffffff-white
  • rgb-red green blue
    1. range (0-255)
    2. define "rgb(0,0,255)"
  • rgba - red green blue alpha
    1. range of colors (0-255)
    2. range of alpha (0.0-1.0)
    3. define "rgba(0,0,255,0.5)"
  • hsl - hue saturation lightness
    1. hue - (0-360)
      • 0-red
      • 120-green
      • 240-blue
    2. saturation - (0-100%)
    3. lightness - (0-100%)
    4. define - "hsl(120,50%,50%)"
  • hsla - hue saturation lightness alpha
    1. hue - (0-360)
      • 0-red
      • 120-green
      • 240-blue
    2. saturation - (0-100%)
    3. lightness - (0-100%)
    4. alpha - 0.0-1.0
    5. define - "hsla(120,50%,50%,1.0)"
