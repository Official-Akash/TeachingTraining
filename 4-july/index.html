Regular Expressions

  • Defines Sequence of Characters
  • Simple patterns as well as complex patterns
  • Syntax: /pattern/modifier
  • Usage: /Akash/i

Modifiers

  1. i --> for case-insensitive
  2. g --> for global match (all in one line)
  3. m --> for multiline matching

Pattern Formats

  1. [axj] --> for any character in brackets
  2. [A-Z] --> for specified range
  3. (a|b) --> for alternatives

Methods

    • test() --> for checking pattern case (returns true/fasle)
    • Usage: /a/i.test("string a")
    • exec() --> for checking pattern case (returns pattern)
    • Usage: /a/i.exec("string a")

JSON - JavaScript object Notation

  • JSON.parse() -- converts a string into JSON object
  • JSON.stringify() -- converts JSON object to a string
  • arrays in JSON

In JSON data is represented as "name":"value" pair.
But in JavaScript it can be written as or defined as name:"value".

Cookies

  • Using JavaScript
  • Cookies are data, stored in small text files, on your computer.
  • Store data as name=value pair
  • Cookie can be Created as: document.cookie = "name=Akash";
  • More parameters document.cookie = "name=Akash; expires=tue 03 july 2018 16:00:00 UTC; path=/;";
  • Cookie can be retrieved as var x = document.cookie;
